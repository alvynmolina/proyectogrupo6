Ayuda de Windows

Todos necesitamos ayuda de vez en cuando. Nadie puede recordar todos los trucos para trabajar con Windows y mucho menos sobre todos los demás programas de su computadora. Además, las cosas cambian con frecuencia de una versión a la siguiente. Los trucos y métodos que aprendió puede ser que ya no funcionen. La tendencia actual, es poner todos los recursos para contestar a sus preguntas directamente en la computadora, en lugar de ser impresas.

  • Buscar una respuesta
  • Preguntar a alguien
  • Información de Microsoft
 

Buscar una respuesta

Windows Fundamentos: Aqui se puede encontrar respuestas a las preguntas más fundamentales sobre nuestro equipo, Windows y algunos de sus componentes principales, tales como el Escritorio, Menú Inicio, Trabajo con ventanas y Uso de botones, menús, etc.

Seguridad y mantenimiento: Aqui se encuentra una buena ayuda sobre temas relativos a la seguridad y el mantenimiento de nuestro equipo, como puede ser el Centro de seguridad, el Control de cuentas de usuarios, Windows Defender o el Control parental, entre otros más.

Ayuda en pantalla de Windows: Esta opcion lleva directamente a la Ayuda en pantalla de Microsoft (pero para ello se necesita estar conectado a Internet).

Índice: Aqui se encuentra un índice general de la Ayuda de Windows

Solucionador de problemas: Aqui se tiene un rápido acceso a las respuestas a los principales problemas que se nos pueden presentar en el uso de Windows, tales como problemas con la Red, con Internet, con el correo electrónico o con el hardware y controladores.

Novedades: Aqui se ofrece un repaso a las principales novedades de nuestra versión de Windows
